# Changelog

All notable changes to this project will be documented in this file.

## 0.1.0

Initial public release.

- Added the IBM i DB2 n8n community node.
- Added credentials for IBM Toolbox JDBC connection options.
- Added Select, Execute SQL, Update, Insert, Insert List, Batch Update, Upsert, and metadata operations.
- Added named and positional SQL parameter support.
- Added transaction modes for write operations.
